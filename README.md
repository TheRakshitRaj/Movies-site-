# 🎬 Movie Website

A responsive and interactive **movie website** built using **HTML, CSS, and JavaScript**.
This project allows users to explore movies, view details, and enjoy a clean and dynamic user interface.

---

## ✨ Features

* 🎥 Browse a collection of movies
* 🔍 Search functionality for movies
* 📄 View movie details (title, rating, overview, etc.)
* 📱 Fully responsive design (mobile + desktop)
* ⚡ Fast and lightweight (no frameworks used)
* 🎨 Clean UI with modern styling

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & layout
* **JavaScript (Vanilla JS)** – Functionality & interactivity
